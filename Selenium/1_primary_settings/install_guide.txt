ГАЙД по установке selenium, доп. пакетов и настройке виртуального окружения.

1.	Открыть пайчарм и обновить менеджер загрузки пакетов pip:
    > python.exe -m pip install --upgrade pip (или python3 -m pip install --upgrade pip)

2.	Создать новый пустой проект (напр. AutomatizationCourse) — На ПК в папке D:\PycharmProjects вручную создать новую папку с таким
    названием -> открыть пайчарм и выбрать/перейти в эту папку проекта.

3.	Создать и активировать виртуальное окружение.
    (Вообще, если через пайчарм создать новый проект, то оно создастся автоматически, но мы должны это уметь делать сами).
    Предварительно деактивировать имеющееся вирт. окружение (но лучше не трогать вирт. окр., если мы работаем в Pycharm):
    > deactivate
    Затем удалить его папку.
    Создать новое вирт. окружение и перейти в него:
    > mkdir environments
    > cd environments
    > python3 -m venv selenium_env (имя можно дать любое, напр., просто venv)
    > selenium_env
    Активировать созданное вирт. окружение:
    > Scripts\activate (или Scripts\activate.ps1)
    (!) ВАЖНО: команда selenium_env\Scripts\activate.bat работает только в командной строке.
    (!) ВАЖНО: окружение работает в рамках текущей сессии окна терминала, его нужно активировать каждый раз.
    Если мы закроем терминал или пайчарм, то окружение потом снова нужно будет активировать.

5.	Установить Selenium:
    > pip3 install selenium

6.	На всякий случай еще установить библиотеку packaging:
    > pip3 install packaging

7.	Установить библиотеку webdriver-manager:
    > pip3 install webdriver-manager

8.  Создать файл.py и в нем инициализировать  Chrome драйвер (см. файл init driver.py)
