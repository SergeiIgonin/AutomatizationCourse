СТАУСЫ ФАЙЛОВ И ОСНОВНЫЕ КОМАНДЫ GIT

Созданный файл — статус "untracked"
git add — статус "staged" (подготовленный/отслеживаемый)
git commit — статус "commited" (зафиксированный - по аналогии сохранения в компьютерной игре)
Измененный файл — статус "modified"

git status — узнать текущий статус репозитория
git add — подготовить файлы к коммиту (здесь есть много различных способов/дополнений к этой команде(что и как именно добавлять))
git commit -m "Сообщение снимка/слепка локального репозитория" — сделать коммит
git push — сделать пуш слепка локального репозитория в глобальный репозиторий
git log — перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке — последние коммиты находятся вверху
