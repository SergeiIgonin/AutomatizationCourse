СТАУСЫ ФАЙЛОВ И ОСНОВНЫЕ КОМАНДЫ GIT

Созданный файл имеет статус "untracked"
Измененный файл имеет статус "modified"

Стандартный флоу пуша файлов в репозиторий GitHub:
1. (.venv) cd C:\*\*                # перейти в корневую папку проекта в вирт. окружении (или в нужную нам папку)
2. git add                          # подготовить файлы к коммиту, сделать их отслеживаемыми ("staged")
3. git commit -m "Custom message"   # сделать снимок файлов/зафиксировать их в локальном репозитории ("commited")
4. git push                         # сделать пуш снимка локального репозитория в глобальный репозиторий

Дополнительные команды:
git status                          # узнать текущий статус репозитория
git log                             # отображает все коммиты в обратном хронол. порядке (свежие коммиты будут вверху)
