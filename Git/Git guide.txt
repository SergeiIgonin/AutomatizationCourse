СТАУСЫ ФАЙЛОВ И ОСНОВНЫЕ КОМАНДЫ GIT

Созданный файл имеет статус "untracked"
Измененный файл имеет статус "modified"

Ручной (упрощенный) флоу создания удаленного репозитория и пуша файлов в него:
Создать проект в Pycharm
Создать одноименный репозиторий на GitHub
git add . или git add *имя файла/папки*   # подготовить добавленные/измененные файлы к коммиту, сделать их отслеживаемыми ("staged")
git commit -m "first commit"              # сделать очередной коммит
git push                                  # сделать пуш снимка локального репозитория в глобальный репозиторий

Тот же флоу, но полностью через терминал:
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/SergeiIgonin/*имя проекта*.git
git push -u origin main

Дополнительные команды:
git status                          # узнать текущий статус репозитория
git log                             # отображает все коммиты в обратном хронол. порядке (свежие коммиты будут вверху)
git init
